This is an implementation of Ticket to Ride game.

Game rules are available in tt_rules_en.pdf and tt_rules_shortened.pdf

More information about the game can be found in https://en.wikipedia.org/wiki/Ticket_to_Ride_(board_game)

This project is implemented during AI master course.

ttr.pl is the master file which initiates the game. All users have to be defined at the beginning of this file.
